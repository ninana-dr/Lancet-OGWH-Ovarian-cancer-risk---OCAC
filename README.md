# Lancet-OGWH-Ovarian-cancer-risk---OCAC
Example R scripts for multiple imputation using MICE, including imputation diagnostics and analysis workflows for an individual participant data pooled analysis. 
Original IPD cannot be shared because of study-specific data sharing agreements.
This repository provides the exact imputation workflow.
The code was implemented using MICE version 4.4.2.
Users can apply the script to data with analogous variable names.
